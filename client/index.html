Flickr Community Connections

With information visualization as our goal, we decided to focus on how the Flickr community groups are connected by the many users across the website. In order to accomplish this, we are using a downsampled version of the Flickr Creative Community dataset curated and provided by Flickr.

Using the provided Flicker user data, we extracted all of the groups from the user records. These groups are represented by nodes. We then computed the users that are shared between any two groups. These connections are represented by edges, and the edge strength is the number of shared users.

We then clustered groups together that shared the same words in the title. For example, a group called "New Airplanes" and a group called "Fast Airplanes" would be clustered by "Airplanes". Due to the scope of this project, the represented groups and clusters make up the most popular groups within Flickr. Over the entire set of groups there are over 200,000 connections between them all. We believe this effectively shows the vast web of interaction users experience as almost all groups have some connections.
